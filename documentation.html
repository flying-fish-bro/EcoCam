Getting Started

Welcome to EcoCurrent! This guide will walk you through everything from unboxing to your first kilowatt-hour generated.

💡
Quick Start: Most installations are complete in under 4 hours. You'll need your SmartLink Hub, the EcoCurrent app, and a Wi-Fi connection.

What's in the Box

  • EcoCurrent Gen 3 Solar Panels (quantity per plan)
  • SmartLink Hub + power adapter
  • Mounting hardware kit
  • Connection cables (colour-coded)
  • Quick-start guide card

Installation

Follow these steps to mount and wire your EcoCurrent panels safely. If you prefer, our certified installer network can handle the entire process.

⚠️
Safety First: Always disconnect your home from the grid before connecting the inverter. EcoCurrent is not responsible for installations not following these guidelines.

Step 1 – Mount the Rails

Using the included L-brackets and lag bolts, mount the aluminium rails to roof rafters spaced no more than 1.2 m apart. Ensure rails are level to within 2 mm across their full length.

Step 2 – Attach Panels

Slide the mid-clamps into the rails, position each panel, and tighten the clamps to 20 Nm. End-clamps secure the outermost panels. Leave a 10 mm expansion gap between adjacent panels.

Step 3 – Wire the Array

Connect panels in series using the MC4 connectors (male → female, following the colour coding). Run the combined positive and negative DC leads to the inverter location.

# Example series string for 10 panels @ 40V each
# Total open-circuit voltage = 400V DC
Panel_1(+) → Panel_2(-) → ... → Panel_10(+) → Inverter DC Input

Step 4 – Connect the Inverter

Mount the EcoCurrent inverter in a shaded, ventilated location within 5 m of the main electrical panel. Connect DC inputs, then AC output to a dedicated 30A breaker.

SmartLink Hub

The SmartLink Hub is the brain of your EcoCurrent system. It monitors output, manages battery charging, and connects to the cloud for AI-powered forecasting.

ℹ️
First boot: The hub broadcasts a temporary Wi-Fi network called ECO_SETUP_XXXXXX. Connect to it with the EcoCurrent app to complete onboarding.

Local Network Access

The SmartLink Hub exposes a local REST API on port 8080 for integrations with home automation platforms like Home Assistant or Apple HomeKit.

API Reference

The EcoCurrent REST API lets you read live data, control charging schedules, and receive webhook events. All endpoints require an API key generated in the app under Settings → Developer.

Authentication

GET https://api.ecocurrent.io/v1/status
Authorization: Bearer YOUR_API_KEY

GET /v1/status

Returns the current system status including power generation, battery level, and grid draw.

{
  "generation_w": 3420,
  "battery_pct": 87,
  "grid_draw_w": 0,
  "mode": "solar_only",
  "timestamp": "2026-03-07T14:22:00Z"
}

POST /v1/schedule

Set a custom charging schedule. The body accepts an array of time-windows and target battery percentages.

POST /v1/schedule
Content-Type: application/json

{
  "windows": [
    { "start": "22:00", "end": "06:00", "target_pct": 100 },
    { "start": "12:00", "end": "14:00", "target_pct": 80 }
  ]
}
🔧
Rate limits: Free-tier keys are limited to 60 requests/minute. Pro and Commercial keys have unlimited API access.
